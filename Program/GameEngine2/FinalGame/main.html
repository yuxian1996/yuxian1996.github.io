Game Engine2 - Final Game


Game Description

The game will have some cube meshes and different color shaders to present different units. Player can control a unit and move it around and jump. If the unit goes on top of another unit, the underneath unit disappears and plays a sound effect. If two units collide in air, both will bounce back. There would be AI to control other units. There would be background music.

Progress

So far I’m able to add the audio system to my final game and play background music. The camera view is set to be 45 degrees top down. However I had some problems to integrate the collision system into my project. Since the project handles all the collision and physic simulation by itself, it doesn’t provide any interface to let use know if any object is colliding or will be collided with other object in this frame. I’m trying to extract the collision detection part from the project and send a collision event to my game. Or simply use sphere collision instead of cube if I can’t extract it successfully.

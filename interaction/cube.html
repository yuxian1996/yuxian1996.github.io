CUBE

Abstract

CUBE is an adventure game. The player can control a cube to roll, jump, rotate, using keyboard or the controller we made with Arduino. The player can pass the level only when they react correctly and find the right way.The player need control the cube to find something and discovery different worlds represented of different human emotions. The main purpose of this game is to encourage people to persuading their true emotions.

Instruction

1) The game runs on windows platform. Decompress “Cube.zip” and run “cube.exe” under the folder.
2) Interaction
Press up, down, left, right arrow to roll, space to jump, Z/X to rotate camera, U to begin a dialog, A/D to move when sliding, ESC to return to the main scene.
